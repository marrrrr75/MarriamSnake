# MarriamSnake (Pygame)

## О проекте
Классическая игра "Змейка". Игрок управляет змейкой, собирает яблоки и старается не столкнуться с границами экрана или самим собой.  
Проект подходит для изучения Pygame и создания небольших игр на Python.

## Функциональность
- Управление змейкой с клавиатуры (стрелки или WASD)
- Появление яблок в случайных местах
- Подсчет очков
- Перезапуск после смерти при нажатии пробела
- Увеличение скорости по мере роста змейки

## Требования / зависимости
- Python 3.10+
- Git (для клонирования репозитория)
- Библиотеки:
  - [Pygame](https://www.pygame.org/news)

## Установка и запуск проекта

## Клонируем репозиторий:
```bash
git clone git@github.com:marrrrr75/MarriamSnake.git
```
```bash
cd MarriamSnake
```

## Создаём виртуальное окружение:
```bash
python3 -m venv venv
```

## Активируем виртуальное окружение:
```bash
source venv/bin/activate
```

## Обновляем pip (рекомендовано):
```bash
python3 -m pip install --upgrade pip
```
## Устанавливаем зависимости проекта:
```bash
python3 -m pip install -r requirements.txt
```

## Запускаем игру:
```bash
python3 snake-game-src/snake.py
```
