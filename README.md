# MarriamSnake (Pygame)

## О проекте
Классическая игра "Змейка" . Игрок управляет змейкой, собирает яблоки и старается не столкнуться с границами экрана или самим собой

## Функциональность
- Управление змейкой с клавиатуры (стрелки или WASD)
- Появление яблок в случайных местах
- Подсчет очков
- Перезапуск после смерти при нажатии пробела
- Увеличение скорости по мере роста змейки
